## Recent Updates

Stay current with Personal's evolution. You receive automatic updates, but review migration notes for breaking changes.

<Update label="2024-10-15" description="v1.3.0" tags={["feature", "improvement"]}>

## New Features

- Added real-time collaboration on documentation pages, enabling multiple users to edit simultaneously.
- Introduced version history with diff viewer for tracking changes over time.

## Improvements

- Enhanced search functionality with fuzzy matching and full-text indexing for faster results.
- Optimized page load times by `>30%` through improved asset bundling.

## Bug Fixes

- Fixed rendering issues with nested lists in Markdown editor.
- Resolved authentication errors during session expiration.

</Update>

<Update label="2024-09-20" description="v1.2.0" tags={["feature", "bugfix"]}>

## New Features

- Integrated webhook support for custom notifications on document updates.
- Added export options for PDF and Markdown formats directly from the dashboard.

## Bug Fixes

- Corrected sidebar navigation glitches on mobile devices.
- Patched potential XSS vulnerabilities in user-generated content.

</Update>

<Update label="2024-08-10" description="v1.1.0" tags={["breaking", "feature"]}>

## Breaking Changes

- Updated API authentication to use JWT tokens instead of API keys. Migrate by regenerating tokens at `https://dashboard.example.com/settings/tokens`.

## New Features

- Launched customizable themes with your brand color `#3B82F6`.
- Added API endpoints for programmatic document management.

## Improvements

- Streamlined onboarding with interactive tutorials.

</Update>

## Update Process

Follow these steps to apply updates manually if needed.

<Steps>
  <Step title="Check Current Version" icon="check-circle">

    Run this command in your project directory:

    ````bash
    personal --version
    ````

  </Step>
  <Step title="Backup Data" icon="database">

    Export your documentation:

    ````bash
    personal export --output ./backup
    ````

  </Step>
  <Step title="Install Update" icon="download">

    <Tabs>
      <Tab title="npm" icon="package">
        ````bash
        npm install personal@latest
        ````
      </Tab>
      <Tab title="Yarn" icon="package">
        ````bash
        yarn add personal@latest
        ````
      </Tab>
    </Tabs>

  </Step>
  <Step title="Verify" icon="zap">

    Test core functionality:

    ````bash
    personal serve --port 3000
    ````

    Visit `http://localhost:3000` and confirm everything works.

  </Step>
</Steps>

## Migration Notes

<Callout kind="alert">

For v1.1.0, replace API keys with JWT tokens. Update your requests:

</Callout>

<CodeGroup tabs="JavaScript,cURL">
  ````javascript
  // Old API key auth
  const response = await fetch('https://api.example.com/docs', {
    headers: { 'Authorization': 'Bearer YOUR_API_KEY' }
  });

  // New JWT token auth
  const token = await getJWTToken(); // Implement your token fetch
  const response = await fetch('https://api.example.com/docs', {
    headers: { 'Authorization': `Bearer ${token}` }
  });
  ````

  ````bash
  # Old
  curl -H "Authorization: Bearer YOUR_API_KEY" https://api.example.com/docs

  # New
  curl -H "Authorization: Bearer YOUR_JWT_TOKEN" https://api.example.com/docs
  ````
</CodeGroup>
