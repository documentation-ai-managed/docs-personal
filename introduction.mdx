## Overview

Welcome to Personal, your dedicated documentation space. You can organize, manage, and collaborate on all project documentation in one place. Personal provides a rich editing experience with version control, search, and sharing capabilities tailored for teams and individual developers.

Personal eliminates scattered notes and outdated wikis by centralizing everything. Create pages, embed code snippets, and track changes effortlessly.

## Quick Start

Get up and running in minutes.

<Steps>
  <Step title="Create Account" icon="user-plus">
    Sign up at `https://app.personal.com` with your email.
  </Step>
  <Step title="New Workspace" icon="plus">
    Create a new documentation space for your project.
  </Step>
  <Step title="First Page" icon="file-plus">
    Click "New Page" and start writing with markdown support.
  </Step>
  <Step title="Publish" icon="globe">
    Share your docs via public links or embed them.
  </Step>
</Steps>

## Embed Examples

Integrate Personal docs into your apps.

<CodeGroup tabs="JavaScript,Python">
  ```javascript
  // Embed a Personal page
  embedPersonalDoc('doc-id-here', {
    theme: 'light',
    height: '500px'
  });
  ```
  ```python
  # Python embed via iframe
  iframe_src = "https://app.personal.com/embed/doc-id-here"
  print(f'<iframe src="{iframe_src}" height="500"></iframe>')
  ```
</CodeGroup>
