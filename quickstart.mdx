## Prerequisites

Before starting, ensure you have:

- A modern web browser like Chrome, Firefox, or Safari
- A valid email address for account verification
- Optional: Existing documentation files (Markdown, GitHub repo) to import

</Callout>

## Create Your Account

Sign up for Personal in under 2 minutes.

<Steps>
  <Step title="Visit Signup Page" icon="external-link">
    Open your browser and navigate to `https://app.personal.com/signup`.
  </Step>
  <Step title="Enter Details" icon="user-plus">
    Provide your email address and create a strong password. Click **Create Account**.
  </Step>
  <Step title="Verify Email" icon="mail">
    Check your inbox for a verification email from Personal. Click the link to activate your account.
  </Step>
  <Step title="Log In" icon="log-in">
    Return to `https://app.personal.com/login`, enter your credentials, and access your dashboard.
  </Step>
</Steps>

## Create Your First Documentation Space

Once logged in, set up a space for your project docs.

<Steps>
  <Step title="New Space" icon="plus">
    From the dashboard, click **New Space** in the top-right corner.
  </Step>
  <Step title="Name It" icon="edit-3">
    Enter a name like `My Project Docs` and select a workspace (or create one).
  </Step>
  <Step title="Customize" icon="settings">
    Choose a description, add a brand color (e.g., `#3B82F6`), and click **Create**.
  </Step>
</Steps>

Your new space appears in the sidebar. Click it to open the editor.

## Import Existing Documentation

Bring in your current docs seamlessly.

<Tabs>
  <Tab title="Markdown Files" icon="file-text">
    <Steps>
      <Step title="Upload">
        In your space, click **Import** > **Files**. Select `.md` or `.mdx` files.
      </Step>
      <Step title="Review">
        Personal auto-generates structure. Edit paths and frontmatter as needed.
      </Step>
    </Steps>
  </Tab>
  <Tab title="GitHub Repo" icon="github">
    Connect your repo directly.

    <CodeGroup tabs="CLI,Web">
      ````bash
      # Install Personal CLI (optional)
      npm install -g @personal/cli

      # Import repo
      personal import my-repo https://github.com/user/my-docs.git --space="My Project Docs"
      ````

      ````mdx
      1. Go to **Import** > **GitHub**
      2. Authorize Personal
      3. Paste repo URL: `https://github.com/user/my-docs`
      4. Select branch and import
      ````
    </CodeGroup>
  </Tab>
</Tabs>

<Callout kind="tip">

After importing, review pages for custom components. Use the search bar to find and edit specific files.

</Callout>
